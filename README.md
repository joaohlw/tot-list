Aplicativo de lista de tarefas desenvolvido em React Native com Expo, com persistência local usando AsyncStorage.

Pré-requisitos

Node.js (versão 14 ou superior) npm ou yarn Expo Go app no celular (Android ou iOS)

Instalação Clone o repositorio

npm install npx expo install react-native-web react-dom npx expo start
npx expo install @react-native-async-storage/async-storage

npx expo start
